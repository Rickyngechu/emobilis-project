Porpular resources

Figma

A cloud-based design tool that offers a wide range of features for UI design, including vector editing, prototyping, and collaboration.
Framer

A powerful UI design tool that is based on React Native, allowing you to create interactive prototypes and deploy them to mobile devices.

Adobe XD

A comprehensive design tool from Adobe that includes wireframing, prototyping, and interaction design capabilities.

BootStrap

Bootstrap is a free and open-source CSS framework directed at designing responsive, mobile-first projects. It provides a wide range of pre-built components and styles that can be used to create a variety of websites and web applications.

Tailwind css

Tailwind CSS is a utility-first CSS framework. It provides a large collection of low-level utility classes that can be used to style HTML elements.

Materialise

Materialize is a CSS framework based on Google's Material Design guidelines. It provides a set of components and styles that can be used to create modern and responsive web applications with a consistent look and feel.

Browser stack

A cloud-based testing platform that allows you to test your website or app on a wide variety of real browsers and devices.

Postman

This is basically a tool for testing and debugging APIs for development and deployment purposes.

Heroku

Heroku is a cloud platform for deploying and hosting Ruby on Rails applications. It is known for its ease of use and its support for a variety of add-ons.

Vercel

Vercel is a cloud platform for deploying and hosting web applications. It provides a variety of features, including a simple deployment process, automatic scaling, and a global CDN.

Netlify

Netlify is another popular cloud platform for deploying and hosting web applications. It is known for its ease of use and its support for a wide range of frameworks and CMSs.
Visual Studio Code

A popular open-source code editor that supports a wide variety of languages and extensions.
Sublime text

This is another popular code editor that is known for its speed and customization.
